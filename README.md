# Smart-Trolley-Billing-System
"An IoT-based smart shopping trolley using ESP32, RFID, and weight sensors for automated billing and QR payments."
Smart Trolley Billing SystemAn IoT-based smart shopping trolley designed to revolutionize the retail experience. By leveraging the ESP32, RFID technology, and load cell weight sensors, this system automates the checkout process, eliminates long queues, and allows shoppers to pay instantly via QR code directly at the trolley.🚀 FeaturesAutomated Item Detection: Instantly scans items added to the cart using RFID tags.Anti-Theft / Weight Verification: Cross-checks the RFID-scanned item against its actual weight using a load cell to prevent billing fraud.Real-time Billing Display: Displays the current item list, individual prices, and the total bill on a local LCD screen.Seamless QR Payments: Generates a dynamic QR code (via display or mobile integration) for instant checkout using UPI or digital wallets.Cloud Connectivity: Sends purchase data to a central database or mobile app via ESP32's built-in Wi-Fi.🛠️ Hardware RequirementsComponentDescriptionESP32 Development BoardThe main microcontroller with Wi-Fi & Bluetooth capabilities.MFRC522 RFID ReaderScans RFID tags attached to products.RFID Tags (13.56 MHz)Attached to items to store unique product IDs.HX711 Amplifier & Load CellMeasures the weight of items in the trolley for verification.I2C 16x2 LCD DisplayDisplays the cart status, item names, and total price.BuzzerProvides audio feedback for successful scans or weight mismatches.Power Supply5V Power bank or rechargeable lithium-ion batteries.💻 Software & TechnologiesLanguage: C++ (Arduino IDE)Protocols: SPI (RFID), I2C (LCD), Serial (HX711)Cloud/Backend: Firebase / MQTT (Optional - for mobile app sync)Libraries Used:MFRC522 (RFID Reader)HX711 (Weight Sensor)LiquidCrystal_I2C (Display)WiFi.h & HTTPClient.h (Cloud sync and QR generation)🔌 Circuit Connection Diagram💡 Note: Double-check your pin configurations in the code before powering up the ESP32.1. 📦 Installation & SetupClone the Repository:Bashgit clone https://github.com/yourusername/Smart-Trolley-Billing-System.git
Set up Arduino IDE:Install the ESP32 Board URL in Preferences.Install the required libraries (MFRC522, HX711, LiquidCrystal_I2C) via the Library Manager.Calibrate the Weight Sensor:Run a basic HX711 calibration sketch to find your load cell's calibration factor and update it in the main code.Configure Wi-Fi:Open Smart_Trolley.ino, find the Wi-Fi credentials placeholders, and insert your network SSID and Password.Upload the Code:Connect your ESP32 to your computer, select the correct COM port, and hit Upload.🔮 Future EnhancementsMobile App Integration: A dedicated Flutter/React Native app to view the cart live on a smartphone.Automatic Weight Calibration: Machine learning integration to adjust for different item distributions inside the cart.Smart Recommendations: Suggesting complementary products on the LCD screen based on current cart items.📄 LicenseThis project is licensed under the MIT License - see the LICENSE file for details.
